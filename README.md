# 版画風画像加工処理LINEBOT 
AWS Lambdaを使ったサーバレス画像加工LINEBOT。 ユーザがこのBOTに画像を送信すると版画風に加工処理された画像を返信する。
※ 現在はLINE BOT APIのトライアル版終了に伴って公開中止。
